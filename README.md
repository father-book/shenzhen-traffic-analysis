## 📊 项目概述

本项目基于**深圳市政府开放数据平台**的交通监控数据（API接口），构建了完整的大数据分析平台，实现了从**数据采集 → 数据仓库建设 → 深度分析 → 可视化展示**的全流程。

### 项目规模
| 指标 | 数值 |
|------|------|
| 总数据量 | 5,544,000 条 |
| 监测街道 | 75 个 |
| 数据时间跨度 | 2018年3月 - 2018年11月 |
| 集群节点 | 4 台虚拟机 |
| 数据存储 | HDFS (约500MB) |

## 🚀 快速开始

### 环境要求
| 组件 | 版本要求 |
|------|----------|
| Hadoop | 3.x |
| Hive | 3.x |
| Spark | 3.x |
| MySQL | 5.7+ |
| Python | 3.6+ |
| JDK | 1.8 |

### 1. 克隆项目
```bash
git clone https://github.com/your-username/shenzhen-traffic-analysis.git
cd shenzhen-traffic-analysis
2. 建表
bash
# 创建数据库
hive -f sql/01_create_database.sql

# 创建ODS层表
hive -f sql/02_ods_table.sql

# 创建DWD层表
hive -f sql/03_dwd_table.sql

# 创建ADS层表
hive -f sql/04_ads_tables.sql
3. 上传数据
bash
# 上传原始数据到HDFS
hdfs dfs -put data/sample_data.csv /user/traffic/raw/
4. 运行分析
bash
# Spark深度分析
spark-submit --master yarn \
    --executor-memory 2g \
    --num-executors 4 \
    scripts/spark_deep_analysis.py

# 执行SQL查询
hive -f sql/05_analysis_queries.sql
5. 查看结果
bash
# 查看ADS表数据
hive -e "SELECT * FROM traffic.ads_hourly_detailed LIMIT 20;"

# 导出结果
hive -e "SELECT * FROM traffic.ads_block_comprehensive;" > results.csv
